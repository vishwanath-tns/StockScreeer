launch_fno_feed.py / fno_launcher.py:

  --force
    Run outside market hours (for testing)
    Default: Only runs 9:15 AM - 3:35 PM IST

  --mode {TICKER, QUOTE, FULL}
    Feed mode (default: QUOTE)
    - TICKER: LTP only (low bandwidth)
    - QUOTE: LTP + depth + OHLC
    - FULL: Complete market data

  --no-futures
    Skip Nifty/BankNifty futures

  --no-nifty-options
    Skip Nifty options (useful to test BankNifty only)

  --no-banknifty-options
    Skip BankNifty options (useful to test Nifty only)

  --include-commodities
    Add MCX commodity futures (Gold, Crude, etc.)

  --debug
    Enable debug logging


fno_db_writer.py:
â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€

  --debug
    Enable debug logging (shows every quote buffered)

  --batch-size N
    Batch size for database writes (default: 100)
    Larger = fewer writes, more memory
    Smaller = more writes, lower latency

  --flush-interval T
    Flush buffer every T seconds (default: 1.0)
    Smaller = more frequent writes
    Larger = fewer writes, risk of data loss on crash
