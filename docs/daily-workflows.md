## üìä Data Management

### Download Specific Stocks

```python
# In Python
from yahoo_finance_service.bulk_stock_downloader import download_stock
download_stock("RELIANCE.NS", days=30)
```

### Import BHAV Files

```powershell
python sync_bhav_gui.py
```

Or set `BHAV_FOLDER` environment variable and run headless.

### Check Data Completeness

```sql
SELECT 
    symbol,
    COUNT(*) as days,
    MIN(date) as first_date,
    MAX(date) as last_date
FROM yfinance_daily_quotes
GROUP BY symbol
ORDER BY days DESC;
```
