## Database Table

### [[planetary-positions]]

Stores minute-level planetary data for 9 celestial bodies:

| Planet | Description |
|--------|-------------|
| Sun | Solar position |
| Moon | Lunar position |
| Mercury | Communication planet |
| Venus | Financial planet |
| Mars | Energy/conflict |
| Jupiter | Expansion/growth |
| Saturn | Restriction/discipline |
| Rahu | North lunar node |
| Ketu | South lunar node |

**Data Points per Planet:**
- `longitude` - 0-360 degrees
- `sign` - Zodiac sign name
- `degree` - Degrees within sign (0-30)

**Coverage:** 2023-2025 (~1.5M records)
