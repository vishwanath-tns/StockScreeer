## ğŸ“¥ Installation

### 1. Clone Repository

```powershell
git clone https://github.com/vishwanath-tns/StockScreeer.git
cd StockScreeer
```

### 2. Install Dependencies

```powershell
pip install -r requirements.txt
```

Key packages:
- `pandas` - Data manipulation
- `sqlalchemy` - Database ORM
- `pymysql` - MySQL connector
- `yfinance` - Yahoo Finance API
- `python-dotenv` - Environment variables
- `tkinter` - GUI framework
- `matplotlib` - Charts

### 3. Configure Environment

Create `.env` file in project root:

```env
MYSQL_HOST=localhost
MYSQL_PORT=3306
MYSQL_DB=marketdata
MYSQL_USER=your_user
MYSQL_PASSWORD=your_password
BHAV_FOLDER=D:\MyProjects\StockScreeer\bhav_data
```

### 4. Setup Database

```powershell
# Create database
mysql -u root -p -e "CREATE DATABASE marketdata CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"

# Run setup scripts
python setup_scripts/create_nifty500_table.py
python setup_scripts/create_indices_tables.py
```
