## Quick Log Command

```powershell
python log.py <action> <file> <description> <category>
```

### Actions
- `create` - New file/feature
- `modify` - Changed existing
- `fix` - Bug fix
- `delete` - Removed
- `cleanup` - Organization
- `refactor` - Code improvement

### Categories
- `feature` - New functionality
- `bugfix` - Bug fixes
- `cleanup` - Code cleanup
- `docs` - Documentation
- `database` - DB changes
- `refactor` - Refactoring

### Example

```powershell
python log.py create scanner.py "Added new momentum scanner" feature
```
