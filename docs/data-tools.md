## Yahoo Finance Tools

### Bulk Stock Downloader
```powershell
python quick_download_nifty500.py
```
Downloads daily data for all Nifty 500 stocks.

**Features:**
- Parallel downloads
- Automatic retry on failure
- Progress tracking
- Upsert (insert or update)

---

## NSE BHAV Tools

### BHAV Import GUI
```powershell
python sync_bhav_gui.py
```
GUI for importing NSE BHAV copy files.

**Features:**
- Drag & drop folder selection
- Duplicate detection (MD5 checksum)
- Progress logging
- Error handling

### BHAV CLI
```powershell
python data_tools/sync_bhav_cli.py
```
Command-line BHAV importer.

## Index Data

### Import Indices
