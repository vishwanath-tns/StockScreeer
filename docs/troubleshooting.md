## Database Issues

### Cannot Connect to Database

**Error:** `Connection refused` or `Access denied`

**Solution:**
1. Check MySQL is running:
   ```powershell
   Get-Service mysql*
   ```
2. Verify `.env` credentials:
   ```env
   MYSQL_HOST=localhost
   MYSQL_PORT=3306
   MYSQL_DB=marketdata
   MYSQL_USER=your_user
   MYSQL_PASSWORD=your_password
   ```
3. Test connection:
   ```powershell
   mysql -u your_user -p -e "SELECT 1"
   ```

### Table Does Not Exist

**Error:** `Table 'marketdata.xxx' doesn't exist`

**Solution:**
Run setup scripts:
```powershell
python setup_scripts/create_nifty500_table.py
python yahoo_finance_service/create_tables.py
```

## Data Download Issues

### Yahoo Finance Rate Limit

**Error:** `Too many requests`

**Solution:**
- Wait 1-2 minutes between large downloads
- Use smaller batch sizes
- Check if Yahoo Finance is blocking your IP

### No Data Returned

**Error:** Empty results from Yahoo Finance

**Solution:**
1. Check symbol format (should be `SYMBOL.NS` for NSE)
2. Verify market is open
3. Check Yahoo Finance website for outages

## Dashboard Issues

### Dashboard Shows 0/0/0

**Error:** Advance-decline shows zeros

**Solution:**
1. Download recent data:
   ```powershell
   python quick_download_nifty500.py
   ```
2. Check if data exists:
   ```sql
   SELECT COUNT(*) FROM yfinance_daily_quotes 
   WHERE date = CURDATE();
   ```

### GUI Not Opening

**Error:** Tkinter errors

**Solution:**
1. Reinstall tkinter:
   ```powershell
   pip install tk
   ```
2. Check Python installation includes tkinter

## Import Issues

### Module Not Found

**Error:** `ModuleNotFoundError: No module named 'xxx'`

**Solution:**
```powershell
pip install -r requirements.txt
```

### Import Path Issues

**Error:** Cannot import local modules

**Solution:**
Run from project root:
```powershell
cd D:\MyProjects\StockScreeer
python launcher.py
```

## Performance Issues

### Slow Queries

**Solution:**
1. Check indexes exist on key columns
2. Optimize query with EXPLAIN
3. Add missing indexes:
   ```sql
   CREATE INDEX idx_symbol_date ON yfinance_daily_quotes(symbol, date);
   ```

### Large File Downloads

**Solution:**
- Use chunked downloads
- Increase timeout settings
- Download in smaller date ranges
