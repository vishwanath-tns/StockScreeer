## Features

- **Multiple Timeframes:** Daily, Weekly, Monthly
- **Technical Indicators:** Moving averages, RSI, MACD
- **Pattern Overlays:** Cup & Handle, VCP patterns
- **Export:** Save as PNG, PDF

## Libraries Used

- `matplotlib` - Core plotting
- `mplfinance` - Financial charts
- `tkinter` - GUI windows

## Example Usage

```python
from charts.chart_tool import create_candlestick_chart

# Create chart for RELIANCE
create_candlestick_chart(
    symbol="RELIANCE.NS",
    days=90,
    show_sma=[20, 50, 200]
)
```
